[[overview-http-status]]
=== HTTP Status Codes

|===
| Status code | Usage

| `200 OK`
| 요청이 성공적으로 처리되었습니다.

| `201 Created`
| 새로운 리소스가 성공적으로 생성되었습니다.

| `400 Bad Request`
| 잘못된 요청입니다.

| `401 Unauthorized`
| 인증되지 않은 사용자입니다.

| `403 Forbidden`
| 접근 권한이 없습니다.

| `404 Not Found`
| 리소스를 찾을 수 없습니다.

| `500 Internal Server Error`
| 서버 내부 오류가 발생했습니다.
|===
